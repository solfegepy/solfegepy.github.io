# Codec Bench

Browser-only encoding and conversion tools. Input never leaves the browser.

## Tools

- **Base64** — encode and decode Base64 and Base64url.
- **URL** — encode and decode URL components.
- **Query Params** — convert query strings and JSON.
- **JWT** — decode JWT headers and payloads without signature verification.
- **Python → JSON** — convert Python literals and JSON in either direction.
- **Timestamp** — convert Z time, Unix seconds, and Unix milliseconds.

## Local development

```sh
cd ui
pnpm install --frozen-lockfile
pnpm run dev
```

Development binds to `0.0.0.0:$UI_INT_PORT`; `$UI_EXT_PORT` controls host exposure. Playwright preview uses
the renamed `$UI_TEST_INT_PORT` variable. Defaults live in `.devcontainer/.env`.

## Build and test

Run current workflow from repository root:

```sh
make build
make lint
make test
make test-e2e
```

`make build` creates production output in `./docs/`. `make lint` runs Prettier, type checks, ESLint, and Semgrep.
Unit and browser tests stop on first failure.

Conversions run on browser main thread and target text inputs up to 1 MB. Larger-input support should move codec work into Web Workers.

`./docs/` is generated by Astro. Do not edit generated files.
