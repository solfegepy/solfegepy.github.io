Z Time and Unix Timestamp Converter

Convert strict UTC Z time, Unix seconds, and Unix milliseconds.

Source

Target

How to convert timestamps

  1. Choose Z time, Unix seconds, or Unix milliseconds as source.
  2. Enter value and choose distinct target format.
  3. Select Convert, then copy result.

Supported Z time range

Z time accepts strict UTC ISO 8601 from 1970 through 9999.

Timestamp format semantics

Supported timestamp formats
FormatOutput
Z timeYYYY-MM-DDTHH:mm:ss.sssZ
Unix secondsWhole seconds since UTC epoch
Unix millisecondsWhole milliseconds since UTC epoch

Decimal timestamp input is accepted. Precision below one millisecond is truncated; Unix output is always an integer.
