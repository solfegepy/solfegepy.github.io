URL Encode and Decode

Encode or decode URL values:

  • RFC 3986 components: encode reserved characters
  • Full URI: preserves URI structure
  • Form URL encoded: uses + for spaces

Source

Target

How to use

  1. Enter plain text or percent-encoded data.
  2. Choose Plain text and one explicit URL encoding mode.
  3. Select Convert, then copy result.

Choose URL encoding mode

Use RFC 3986 component for one value, Full URI for complete URI syntax, or Form URL encoded for one form value. Selection matters because + means space only in form decoding.

URL encoding comparison

URL encoding mode behavior
ModeSpaceUse
RFC 3986 component%20One component value; reserved delimiters encoded.
Full URI%20Complete URI; syntax delimiters preserved.
Form URL encoded+One form value; literal plus becomes %2B.

Form example: a+b / c~d → a%2Bb+%2F+c%7Ed. For structured name=value&name=value data, use Query Params.
